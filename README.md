**Minimalist, secure, modular authentication plugin for PaperMC.**

Author: **[N1xend](https://github.com/N1xend)**  
Support: **Paper 1.21.1+ | Java 21**

## Features
- ✅ BCrypt password hashing
- ✅ Auto-login (configurable duration)
- ✅ MySQL / SQLite support
- ✅ 12 languages
- ✅ IP account limit protection
- ✅ Session logging with rotation
- ✅ GeoIP blocking
- ✅ Modular architecture

mvn clean package
